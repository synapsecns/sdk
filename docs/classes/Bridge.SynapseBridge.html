buildApproveTransaction

  • buildApproveTransaction(args: {
        amount?: BigNumberish;
        token: string | Token;
    }, gasOptions?: GasOptions): Promise<PopulatedTransaction>

  • Builds an ethers PopulatedTransaction instance for an ERC20 Approve call, approving some amount of a given token to be spent by the Synapse Bridge on its chain. The returned PopulatedTransaction must then be passed to the user via Web3 or some other framework so they can ultimately send the transaction. Should ALWAYS be called before performing any bridge transactions to ensure they don't fail.

    Returns

    Populated transaction instance which can be sent via ones choice of web3/ethers/etc.

    Parameters

    • args: {
          amount?: BigNumberish;
          token: string | Token;
      }
      • Optional amount?: BigNumberish

        Optional, a specific amount of args.token to approve. By default, this function builds an Approve call using an "infinite" approval amount.

      • token: string | Token

        Token instance or valid on-chain address of the token the user will be sending to the bridge on the source chain.

    • Optional gasOptions: GasOptions

      Optional gas price/fee options for the populated transaction.

    Returns Promise<PopulatedTransaction>
